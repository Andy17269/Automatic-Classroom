# 自动关机脚本 (auto_shutdown.py) 部署说明

本脚本用于在 Windows 系统上发送一个关机提醒通知，并根据用户的选择执行关机或延迟关机。

---

### 一、所需环境

1.  **操作系统**:
    *   Windows 10 或 Windows 11。

2.  **Python 环境**:
    *   需要安装 Python。可以从官网 `https://www.python.org/downloads/` 下载。
    *   **重要**: 安装时，请务必勾选 "Add Python to PATH" 选项。

3.  **Python 依赖库**:
    *   脚本需要 `winsdk` 库来发送 Windows 通知。请打开命令提示符 (CMD) 或 PowerShell，运行以下命令进行安装：
        ```shell
        pip install winsdk
        ```

---

### 二、部署方法 (使用 Windows 任务计划程序)

此方法可以精确地设置脚本在每天的特定时间自动、静默地运行。

1.  **打开任务计划程序**:
    *   按下 `Win` 键，搜索“任务计划程序”或 "Task Scheduler"，然后打开它。

2.  **创建基本任务**:
    *   在右侧的“操作”面板中，点击“创建基本任务...”。

3.  **命名任务**:
    *   为任务输入一个名称（例如 `Auto Shutdown Reminder`），然后点击“下一步”。

4.  **设置触发器**:
    *   选择任务何时开始（例如“每天”），然后点击“下一步”。
    *   设置您希望脚本运行的具体时间（例如，晚上 6:30:00），然后点击“下一步”。

5.  **设置操作**:
    *   选择“启动程序”，然后点击“下一步”。

6.  **配置程序和参数 (关键步骤)**:
    *   **程序或脚本**: 点击“浏览”，找到并选择 `pythonw.exe`。
        *   `pythonw.exe` 可以在后台静默执行脚本，不会弹出命令行窗口。
        *   它通常和 `python.exe` 在同一个目录下。您可以在命令提示符中运行 `where pythonw` 来找到它的路径。
    *   **添加参数 (可选)**: 在此输入框中，填入 `auto_shutdown.py` 脚本的**完整路径**。例如：
        `C:\Users\your_username\path\to\your\project\auto_shutdown.py`
        (请将路径替换为您的实际文件路径)

7.  **完成创建**:
    *   点击“下一步”，然后点击“完成”。

部署完成后，任务计划程序将在您设定的时间自动执行该脚本。
